<h4>Minecraft Java Edition Ver.1.21.8 - 1.21.11　　　※二次配布はお控えください</h4><br>

<h2>概要</h2>
<p>・ゾンビ, スケルトン, クリーパー, クモ のスポーン時、低確率で強化された状態でスポーンする<br>
・強化は アイアン, ゴールド, ダイヤモンド の3レベルに分かれている<br>
・レベルはパーティクルによって見分けることができる<br>
・倒すとそのレベルに応じてさまざまなアイテムを入手できる<br>
※ハスク, ストレイ, ボグド も強化の対象</p><br>


<h2>詳細</h2>
<h3>スポーンについて</h3>
<p>・スポナーからスポーンしたモンスターは、原則強化されません<br>
・スポーン場所が「Y>60」か「頭上にブロックが存在する」場合も強化されません<br>
　→ 森ではスポーンしにくくなります
・付近に存在する強化されたモンスターが多い場合、強化に制限がかかります</p><br>

<h3>強化内容について</h3>
<h4>共通</h4>
<p>・ゾンビとスケルトンは強化時、防具を着用した状態でスポーンします<br>
・矢などの射出物を回避しようとします<br>
・攻撃対象が複数いる場合、戦闘中に敵対先を変更する場合があります<br>
・ダメージを受けた際、付近のアイアンゴーレムやオオカミに反撃ダメージを与えます</p>

<h2>導入方法</h2>
<p>1. Uncommon Enemy Datapack.zip をクリックしてから、右に表示される Raw をクリックしてダウンロードします。<br>
