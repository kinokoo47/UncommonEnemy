<h2>強化内容について</h2>
<h3>共通</h3>
<p>・ゾンビとスケルトンは強化時、防具を着用した状態でスポーンします<br>
・体力や攻撃力などの基礎ステータスが強化されます<br>
・矢などの射出物を回避しようとします<br>
・攻撃対象が複数いる場合、戦闘中に敵対先を変更する場合があります<br>
・ボートなどに乗せられた際、降りようとすることがあります<br>
・ダメージを受けた際、付近のアイアンゴーレムやオオカミに反撃ダメージを与えます</p>
